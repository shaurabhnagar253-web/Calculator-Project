# Calculator

## Description

This is a simple and responsive Calculator web application developed using **HTML, CSS, and JavaScript**. The calculator performs basic arithmetic operations such as addition, subtraction, multiplication, division, and percentage calculations. It provides a clean and user-friendly interface with instant results.

## Features

* Addition, Subtraction, Multiplication, and Division
* Percentage (%) calculation
* Decimal number support
* Clear (C) button
* Delete (⌫) button
* Instant result calculation
* Responsive and attractive user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

```
Calculator/
```

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Open `index.html` in any web browser.
4. Start using the calculator.
